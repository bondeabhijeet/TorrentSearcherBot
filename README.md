# Want to deploy this bot yourself?
- Recommended operating system<br>
   ```Linux distribution```
   
- Installing python
  ```
  sudo apt-get install python3
  ```
- Cloning this repo
  ```
  git clone https://github.com/bondeabhijeet/TorrentSearcherBot
  ```
- Navigating inside the repo
  ```
  cd TorrentSearcherBot
  ```
